What is this?

This webpage enables you to select from a variety of registry tweaks, which are then compiled into a single registry (.reg) file. Registry tweaks allow you to modify settings, which are not accessible by default, such as disabling telemetry or resizing the taskbar on Windows 11. While some presets aim to improve performance or privacy, others provide purely cosmetic changes. Further down, I explain the functionality of each preset in detail.

Available presets

Remove bloatware and suggestions

First of all, it is worth noting that this does not remove the preinstalled apps. If you aim to remove all UWP apps (which I strongly advise against, as it breaks important system components), you can achieve this by opening a PowerShell window as Administrator and running Get-AppxPackage -AllUsers | Remove-AppxPackage . Having addressed that, a list of the tweak's functionalities is presented below:

  • Preventing automatic installs of recommended apps

  • Preventing automatic reinstalls of preinstalled apps

  • Disabling tips ans suggestions

  • Disabling online suggestions in settings

  • Disabling all other content delivery options

Disable telemetry

This tweak disables the following data collection and telemetry options:

  • Windows data collection and telemetry

  • App diagnostic data permission

  • App launch telemetry

  • Microsoft Edge telemetry data collection

  • Microsoft Office telemetry

Enable Windows 95 taskbar

There are two versions of this tweak. Make sure to select the appropriate version compatible with your edition of Windows. The tweak does the following:

  • Making taskbar icons small (this option does't exist on Windows 11 by default, so make sure that you are using the correct version)

  • Disabling taskbar grouping

Disable Windows Update

To ensure that the system doesn't break, this tweak only disables automatic updates instead of disabling Windows Update in general. Also note that app updates are not affected by this tweak.

Disable Cortana and Web Search

Chosing this won't remove the cortana app itself, although you can achieve that by running Get-AppxPackage *Microsoft.549981C3F5F10* | Remove-AppxPackage in a PowerShell window. Furthermore it does the following:

  • Disabling the cortana app

  • Disabling cortana on the lockscreen

  • Preventing the search from using using your location

  • Disabling web search
